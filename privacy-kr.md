**시행일: 2026년 5월 6일**
**최종 업데이트: 2026년 5월 6일**

ClassiPic(이하 "본 앱")은 사용자의 개인정보를 매우 중요하게 생각하며, 「개인정보 보호법」 및 관련 법령을 준수합니다. 본 처리방침은 본 앱이 어떤 정보를 수집·이용하며, 사용자가 어떤 권리를 갖는지 안내합니다.

## 2. 처리 원칙: 100% 온디바이스(On-device)

본 앱은 사용자의 사진과 사진 메타데이터를 **분석 목적의 외부 전송 없이** 사용자의 기기 내부에서만 처리합니다. 본 앱은 사용자의 사진을 운영자 또는 제3자의 서버로 업로드하지 않습니다.

## 3. 수집·이용하는 정보

### 3.1 기기 내부에서 접근하는 정보 (외부 전송 없음)

| 정보 | 접근 목적 | 저장 위치 |
|---|---|---|
| 사진 라이브러리(PHAsset) | AI 분류, 앨범 표시, 중복 사진 탐지 | 기기 내부(앱 외부 저장 없음) |
| 사진 위치 메타데이터(EXIF GPS) | 위치 기반 자동 분류 | 기기 내부 |
| 사용자 설정 (UserDefaults) | 환경설정 저장 (분류 기준, 잠금 앨범 ID 등) | 기기 내부 |
| 인앱 결제 영수증 (StoreKit) | Lifetime 이용권 검증 | Apple StoreKit |

### 3.2 광고/측정을 위해 수집되는 정보 (Free 플랜에 한함)
본 앱은 무료 사용자에게 Google AdMob 광고를 표시합니다. 광고 송출 과정에서 다음 정보가 Google로 전송될 수 있습니다.
- 광고 식별자(IDFA, 사용자가 ATT에서 허용한 경우에 한함)
- 대략적 IP 주소(시·국가 수준)
- 기기 종류 / OS 버전 / 앱 버전
- 광고 노출·클릭 이벤트

광고 식별자(IDFA)는 사용자가 iOS의 "App Tracking Transparency"(ATT) 다이얼로그에서 허용한 경우에만 사용됩니다. 거부 시에도 광고는 표시되지만 비개인화 광고로만 표시됩니다.

자세한 Google 처리방침은 다음을 참고하시기 바랍니다.
https://policies.google.com/privacy

### 3.3 자동으로 수집되지 않는 것
본 앱은 다음을 수집·저장하지 않습니다.
- 사용자 계정 / 이메일 / 이름 / 전화번호
- 연락처, 캘린더, 건강 데이터
- 사진 자체 또는 사진의 분류 결과(서버 저장 없음)
- 위치 데이터의 외부 전송

## 4. 보유 및 파기

본 앱이 기기 내부에 저장한 정보(설정값, 분류 결과 캐시 등)는 사용자가 앱을 삭제하면 함께 삭제됩니다. 본 앱은 외부 서버에 사용자 정보를 보유하지 않으므로 별도의 파기 절차는 없습니다.

## 5. 사용자의 권리

사용자는 언제든지 다음을 행사할 수 있습니다.
- iOS "설정 → ClassiPic" 에서 사진 / 위치 접근 권한을 철회
- iOS "설정 → 개인정보 보호 → 추적" 에서 광고 식별자 접근 철회
- 앱 삭제로 모든 로컬 데이터 일괄 삭제
- 위 문의 이메일로 추가 권리 행사 요청
